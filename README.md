# TrafficSimulator

Simple traffic simulation project in C++.

## Build

This project uses standard C++ files. Compile with g++:

```bash
g++ -std=c++17 -O2 -o TrafficSimulator *.cpp
```

## Run

Provide an input file or use `input.txt`:

```bash
./TrafficSimulator input.txt
```
